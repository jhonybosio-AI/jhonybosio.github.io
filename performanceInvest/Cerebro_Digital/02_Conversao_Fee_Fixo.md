# Prompt: Conversão para Fee Fixo (O Fim do Conflito de Interesses)

**Objetivo:** Usar a IA para montar um argumento lógico e matemático que prove para o cliente as desvantagens do modelo tradicional de corretagem (bancos/assessores antigos) e as vantagens financeiras e de transparência do modelo Fee Fixo da Performance Invest.

---

## Como usar
Após receber a carteira atual do cliente ou descobrir onde ele investe, use este prompt para gerar o roteiro da sua reunião ou mensagem de conversão.

---

## 🤖 Prompt para Copiar:

```text
Atue como um Consultor Financeiro fiduciário especializado em demonstrar ineficiências em carteiras de investimentos tradicionais. 

**Situação Atual do Cliente:**
- O cliente atualmente investe no [BANCO/CORRETORA ATUAL, ex: Itaú Personnalité ou bancão tradicional].
- O patrimônio aproximado dele é de [VALOR, ex: R$ 2 Milhões].
- Ele tem o perfil [PERFIL, ex: Conservador/Moderado] e os produtos que mais usa são [PRODUTOS, ex: Fundos de Banco, COE, CDBs com taxas ruins].

**Sua Tarefa:**
Construa um roteiro de conversa (pitch) para convencer este cliente a migrar para o modelo "Fee Fixo" (taxa fixa sobre patrimônio, com devolução de cashback/rebates) na Performance Invest.

**O roteiro deve conter:**
1. **O Choque de Realidade:** Uma explicação simples e didática de como o banco/corretora atual dele ganha dinheiro "escondido" (taxa de administração, spread, rebate de fundos), mostrando o conflito de interesses ("o gerente vende o que dá mais comissão pra ele, não o que é melhor para você").
2. **A Matemática (Simulação):** Uma simulação simples mostrando que esses custos ocultos podem chegar a 1,5% ou 2% ao ano. Calcule quanto isso representa em reais sobre o patrimônio de [VALOR] do cliente por ano.
3. **A Solução (Fee Fixo):** Apresente o modelo da Performance Invest (cobramos apenas uma taxa combinada de X%, sem surpresas). Explique como isso alinha 100% os nossos interesses: só ganhamos mais se o seu patrimônio crescer.
4. **Chamada para Ação:** O fechamento perfeito para convencê-lo a fazer um "Raio-X" gratuito da carteira dele conosco.

O tom deve ser educativo, chocante (ao descobrir os custos ocultos), mas extremamente profissional.
```
